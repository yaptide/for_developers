In demo mode, YAPTIDE runs entirely in the browser.
Geant4 is compiled to WebAssembly and executes locally — no backend, no login, no internet connection needed after the initial load.

This is the setup deployed at [yaptide.github.io](https://yaptide.github.io). The Docker image simply serves the static production build — no backend is included.

## Production build and Docker image

Build the production demo bundle:

```bash
npm run build-demo
```

Build a Docker image that serves the static `build/` output:

```bash
docker build -t yaptide-ui .
```

Run the image locally:

```bash
docker run --rm -d -p 80:80/tcp --name yaptide-ui yaptide-ui
```

The UI is then served at **http://localhost**.

## Stopping
```bash
docker stop yaptide-ui
```

Since the container was started with `--rm`, it is automatically removed when stopped.
